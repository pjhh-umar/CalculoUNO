Prácticas Segundo Examen Parcial

Ejercicio

Sea \(a\in \mathbb{R}^{+}\) arbitrario pero fijo. Utiliza el principio de inducción para probar que para todo \(n\in \mathbb{N}\), se satisface la desigualdad: $$a<(1+a)^n$$.

Ejercicio

Si $A,B \subseteq \mathbb{R}$ son conjuntos no vacios acotados superiormente y $\gamma \in \mathbb{R}^{+}$ arbitrario pero fijo, prueba que los siguientes conjuntos son acotados superiormente.

  1. $A \cup B$
  2. $A \cap B$
  3. $\gamma + B \colon =\left\{\gamma + b\, \big|\,b \in B \right\}$. ¿Podemos concluir lo mismo si $\gamma \leq 0$?
  4. $\gamma \cdot B \colon =\left\{ \gamma \cdot b\, \big|\,b \in B \right\}$. ¿Podemos concluir lo mismo si $\gamma \leq 0$?
  5. $A + B \colon =\left\{ a + b\, \big|\,a\in A, b\in B\right\}$

Ejercicio

Sea $A = \left\{ x\in \mathbb{R}\,\big| \, x \geq 0 \right\}$.

  1. Prueba que $A$ es acotado inferiormente pero no es acotado superiormente.
  2. Prueba que $\inf(A)$ existe y que $\inf(A) = 0$.

Justifica clara y detalladamente tu respuesta.

Ejercicio

Sea $A = \left\{ x\in \mathbb{R}\,\big| \, x > 0 \right\}$.

  1. ¿$A$ es acotado inferiormente?
  2. ¿$A$ es acotado superiormente?
  3. ¿Existe $\inf(A)$ ? En caso de existir darlo.
  4. ¿Existe $\sup(A)$?

Justifica clara y detalladamente tu respuesta.

Ejercicio

Determine si el conjunto $\left\{ x\in \mathbb{R}\,\big| \, x^{2} - x + 1 < 0\right\}$ esta acotado superior o inferiormente. Justificar clara y detalladamente tu respuesta.

Ejercicio

Determine si el conjunto $\left\{ x\in \mathbb{R} \, \big|\, 3x + 5 < 4x -7 \right\}$ esta acotado superior o inferiormente. Justificar clara y detalladamente tu respuesta.

Ejercicio

Sean \(A\) un subconjunto no vacio de \(\mathbb{R}\) acotado superiormente. Si existe $M\in A$ tal que $M$ es una cota superior de $A$, prueba que $\sup(A) = M$.

Ejercicio

Sean \(A\) un subconjunto no vacio de \(\mathbb{R}\) acotado inferiormente. Si existe $m\in A$ tal que $m$ es una cota inferior de $A$, prueba que $\inf(A) = m$.

Ejercicio

Demuestra que el supremo e infimo de un conjunto (en caso de existir) no necesariamente pertenecen al conjunto. (Da un ejemplo)

Ejercicio

Sean $a,b \in \mathbb{R}$ con $a < b$. Prueba que $\inf((a,b)), \sup((a,b))$ existen y que $\inf((a,b)) = a$ y $\sup((a,b)) = b$

Ejercicio

Sean \(A\) un subconjunto no vacio de \(\mathbb{R}\) y $u \in \mathbb{R}$. Prueba que las siguientes proposiciones son equivalentes.

  1. $u$ es una cota superior de $A$
  2. Si $t \in \mathbb{R}$ con $t>u$, entonces $t \not \in A$

Ejercicio

Sean \(A\) un subconjunto no vacio de \(\mathbb{R}\) acotado inferiormente. Si $\inf(A) > 0$ prueba las siguientes proposiciones

  • El conjunto $\tilde{A} := \left\{ \displaystyle{\frac{1}{a}}\, \big|\,a \in A \right\}$ esta bien definido. Esto es, prueba que $a\not = 0$ para todo $a \in A$.
  • $\sup{(\tilde{A})} = \displaystyle{\frac{1}{\inf(A)}}$
